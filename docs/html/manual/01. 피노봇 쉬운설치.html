피노봇 쉬운설치¶

1. 쉬운설치 [권장]¶

  1. 링크에서 피노봇이 설치된 라즈베리파이 os 다운로드

    먼저 아래 링크에서 해당 보드에 맞는 전용 OS를 다운로드합니

    • Raspberry Pi Zero

    • Raspberry Pi 3B+

    • Raspberry Pi 4B

  2. win32diskmanager, pi imager, etcher등으로 sd카드에 설치

  3. SD카드 PinoSet 폴더 이동,

    피노봇의 환경설정은 sd카드의 PinoSet 폴더에서 이루어집니다.

  4. 구글 키 복사

  5. 와이파이 설정

    라즈베리파이 와이파이를 SD카드에서 설정하는 방법입니다.

  6. 부팅

    이제 SD카드를 피노봇의 라즈베리파이에 꼽고 부팅을 기다립니다.

    다음과같이 LCD창에 부팅메세지가 표시됩니다.

    네트워크 연결이 안될경우 다음과같이 네트워크 재시도하는데,
재시도가 반복될경우, 5번의 와이파이 설정을 다시 하거나
HDMI및 키보드마우스를 연결해서 수동으로 와이파이 설정을 진행해야합니다.

  7. 부팅 완료메세지

부팅이 완료되면 다음과 같이 부팅완료 메세지가 표시됩니다.

이제 피노봇은 작동준비가 끝났습니다.

Note.

1. 설치된 라즈비안 os의 id는 pi, password 는 pinobot01입니다
2. 쉬운 연결및 설정을 위해서 VNC와 SSH가 켜저있으나, 보안의 문제가 있음으로,
교육용이 아닌 장기사용시 id, password의 변경이나 , vnc,ssh 기능의 비활성화가 필요합니다.
