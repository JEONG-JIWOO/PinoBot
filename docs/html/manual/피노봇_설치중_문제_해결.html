피노봇_설치중_문제_해결¶

1. 네트워크 확인¶

피노봇 설치및 동작에서 주 문제는 네트워크에서 발생이 됩니다.

라즈베리파이에서는 GUI에서 네트워크를 연결하는것이 가장 안전하고 잘 작동합니다.

1.1 네트워크 관련 문제해결¶

  • 무선 네트워크가 비활성화 된 경우

라즈베리파이 OS 재 설치를 권장합니다.

2. 마이크 및 스피커 오동작¶

  • 인터페이스 (I2C, SPI ) 활성화 여부 체크

  • 사운드 드라이버 확인 명령어 입력

alsamixer

Respeaker 2-mic hat 검색 안된경우,

os 재설치 혹은 클린 재설치 권장
